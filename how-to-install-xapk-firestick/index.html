How to Install XAPK & APKS Files on Fire TV, Firestick & Android TV

last updated: 2026-07-22 · ~6 min read

Fire TV, Firestick, Android TV and Google TV all run a version of Android, so you can sideload an .xapk or .apks app just like on a phone — you just need a way to turn on unknown sources, get the file onto the device, and install split APKs. Here's the whole process, plus the gotchas specific to TV.

Before you start: only sideload apps you trust, and check the app is actually built for a TV. Many phone apps and games install fine but are unusable with a remote or on a big screen.

What you'll need

  • The .xapk / .apks file you want to install.
  • A way to move it onto the TV device (a sideloading/transfer app, or USB).
  • A split-APK installer that runs on the device and works with a remote.

Fire TV / Firestick: turn on "Apps from Unknown Sources"

  1. Open Settings

    From the Fire TV home screen, go to Settings → My Fire TV → Developer options.

  2. Enable Apps from Unknown Sources

    Turn it on for the app you'll use to open files (for example, the Downloader app or your installer). Confirm the warning.

If you don't see Developer options, open Settings → My Fire TV → About and click the device name seven times to unlock it.

Fire TV / Firestick: get the file onto the device

The Firestick has no file browser out of the box, so pick one of these:

  • Downloader app — install it from the Amazon Appstore and enter a direct download URL to fetch the .xapk straight onto the device.
  • Send Files to TV — install it on both your phone and the Firestick (same Wi-Fi), then push the file across wirelessly.
  • USB / network drive — on Fire TV boxes with a USB port, copy the file to a stick and browse to it.

Fire TV / Firestick: install the XAPK

  1. Open your split-APK installer

    Launch SAI+ (or a similar installer that runs on Fire TV).

  2. Select the .xapk file

    Browse to where you saved it and choose it. Use the remote's D-pad to move through the list.

  3. Install and confirm

    The installer unpacks the base APK, split APKs and any OBB data, then Android shows a confirmation — select Install.

  4. Launch it

    Find the app in Your Apps & Channels (you may need to "See All").

Android TV & Google TV

The idea is the same, the menus differ slightly:

  1. Allow unknown sources

    Go to Settings → Apps → Security & restrictions → Unknown sources and enable it for your file app or installer.

  2. Transfer the file

    Use Send Files to TV, a cloud/downloader app, or a USB drive.

  3. Install with a split-APK installer

    Open the file in SAI+ and install as above. OBB data is handled for you.

TV-specific tips

  • Pick a remote-friendly installer. Some installers rely on touch gestures that are painful with a D-pad. Choose one with a plain, focusable list.
  • Check the app is built for TV. Portrait-only phone apps and touch-only games may install but be unusable on a TV.
  • Watch your storage. Firesticks have little free space; large XAPK games with OBB data may not fit.
  • Keep unknown sources tidy. Turn it back off after installing if you like — you can re-enable it any time.

Frequently asked questions

Can you install XAPK files on a Firestick?

Yes — Fire TV/Firestick run Android. Enable Apps from Unknown Sources, move the .xapk onto the device, and install it with a split-APK installer, as long as the app inside supports TV.

How do I move an XAPK file to my Fire TV?

Use the Downloader app to fetch it via a direct URL, "Send Files to TV" to push it from your phone over Wi-Fi, or a USB drive on boxes with a USB port.

Do XAPK games work on Android TV?

Only if the game supports TV hardware and input. It will install, but many phone games aren't designed for a TV screen or controller.

It installed but won't open — now what?

See our XAPK troubleshooting guide. On TV it's usually a storage, OBB, or app-not-built-for-TV issue.
